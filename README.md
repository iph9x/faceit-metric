## FACEIT METRIC

Faceit Metric allows you to see: 

* detailed statistics of all players
* player statistics for the last 10-100 matches
* list of recent matches
* detailed match statistics
* comparing players
* elo / kd / frags charts

[Try it here!](http://faceit-metric.ru/)
